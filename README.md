# Medical Shop Website

## 📌 Project Description
The Medical Shop Website is a simple static website created using HTML and CSS. It is designed to display information about a medical store and the medicines available. The website helps users view medicines, prices, and basic details in an organized and user-friendly interface.

## 🎯 Objective
The main objective of this project is to design a basic medical shop website layout that allows users to easily view information about medicines and medical products.

## 🛠 Technologies Used
- HTML
- CSS

## 💡 About the Website
This website contains multiple sections such as:
- Home page
- About the medical shop
- List of medicines/products
- Contact information

The website layout is designed using HTML for structure and CSS for styling and visual appearance.

## ⚙ How the Website is Developed
1. **HTML** is used to create the structure of the website such as headings, images, product lists, and navigation menus.
2. **CSS** is used to style the website by adding colors, fonts, layouts, and spacing.
3. Different sections are organized to make the website simple and easy to navigate.

## 📂 Website Pages
- **Home Page** – Displays the main introduction of the medical shop.
- **Products Page** – Shows available medicines and medical products.
- **About Page** – Provides information about the medical shop.
- **Contact Page** – Contains contact details for the shop.

## 📚 What I Learned

- Designing and styling web pages using CSS
- Organizing content into different sections
- Improving website layout and user interface
